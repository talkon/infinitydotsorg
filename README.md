### Codes that should normally include at the head of the file

~~~~html
<script src='https://www.infinitydots.org/assets/scripts/csi.min.js'></script>
~~~~

~~~~html
<div data-include="https://www.infinitydots.org/navbar.html"></div>
~~~~


